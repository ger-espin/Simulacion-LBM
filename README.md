Para un proyecto de hidrodinámica simulé el flujo de un fluido incompresible alrededor de un obstáculo cilíndrico.
El objetivo era observar cómo cambia el flujo laminar y el límite para el número de Reynolds en el que el flujo empieza a presentar vórtices.

El programa genera una serie de imágenes del fluido que pueden ser unidas mediante algún programa de edición para hacer un gif.
Además, calcula el número de Strouhal, genera gráficas del Coeficiente de Arrastre (Cd) y del Coeficiente de Sustentación (Ci).
